== Description ==

Simple Social Sharing buttons for WordPress that dosent effect the pageload speeds. You can add Facebook, Twitter, Linkedin, Reddit buttons to your post.


== Installation ==


1. Upload `zoorvy-social-share` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Choose your preference in Settings->Zoorvy Social Share
